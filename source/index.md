## 变量


下面两个是可以在命令行中定义的变量

* theme
* transition

```html
<!DOCTYPE html>
<html lang="en">
<head>
  <meta charset="UTF-8">
  <title>Document</title>
</head>
<body>
  <script type="text/javascript">
    alert(1)
  </script>
</body>
</html>
```
